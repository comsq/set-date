# set-date

Скрипт для подмены даты модификации скриншотов

#### Запуск

1. Положить файл `set_dates.py` в директорию со скриншотами.
<p>Для поиска скриншотов используется регулярное выражение: <em>.*?(\d{4}).(\d{2}).(\d{2}).(\d{2}).(\d{2}).(\d{2})\.png</em></p>

Примеры корректных имён файлов скриншотов:
```bash
2018-10-26_22-13-37.png
Скриншот 2018.10.26.22.13.37.png
____2018 10 26 22 13 37.png
2018_10_26_13_37.png
```

2. Выдать права на запуск:
```bash
chmod +x set_dates.py
```

3. Запустить
```bash
./set_dates.py
```
